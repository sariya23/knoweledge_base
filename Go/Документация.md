[[Go]]
Нужно скачать утилиту `pkgsite`. После этого перейти в директорию с каким-то пакетом и в ней выполнить
```shell
pkgsite -open . 
```
Откроется НАСТОЯЩАЯ документация этого пакета

```go
func ExampleAdd() {
	sum := Add(2, 3)
	fmt.Println(sum)
	// Output: 5
}
```
